# рЃЮрЃюрЃџрЃљрЃўрЃю рЃерЃћрЃАрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

## ­Ъћљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃљрЃЊрЃЏрЃўрЃюрЃў (gigalabichi1) рЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃА рЃљрЃ«рЃљрЃџ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА
- JWT рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃАрЃћрЃАрЃўрЃћрЃЉрЃў
- bcrypt рЃћрЃюрЃЎрЃарЃўрЃърЃбрЃљрЃфрЃўрЃљ рЃърЃљрЃарЃЮрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА

## ­ЪЊі рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў
- рЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃўрЃА рЃбрЃарЃћрЃЎрЃўрЃюрЃњрЃў рЃАрЃљрЃЏрЃў рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃЌ: рЃЊрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў Рєњ рЃњрЃќрЃљрЃерЃўрЃљ Рєњ рЃЕрЃљрЃЏрЃЮрЃАрЃБрЃџрЃў
- рЃЏрЃЮрЃЏрЃгрЃЮрЃЊрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
- рЃўрЃюрЃЋрЃЮрЃўрЃАрЃћрЃЉрЃўрЃА рЃљрЃдрЃарЃўрЃфрЃ«рЃЋрЃљ рЃЌрЃљрЃарЃўрЃдрЃћрЃЉрЃўрЃЌ
- рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў (рЃгрЃЮрЃюрЃљ/рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃљ)
- рЃдрЃўрЃарЃћрЃЉрЃБрЃџрЃћрЃЉрЃўрЃА рЃбрЃарЃћрЃЎрЃўрЃюрЃњрЃў рЃЉрЃўрЃџрЃўрЃА рЃюрЃЮрЃЏрЃарЃћрЃЉрЃўрЃЌ
- Excel рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў

## ­Ъїљ рЃЮрЃюрЃџрЃљрЃўрЃю рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ
- PostgreSQL рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ
- RESTful API real-time рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃћрЃЉрЃўрЃЌ
- рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў
- Multi-user support

## ­Ъџђ рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ
1. Clone: `git clone https://github.com/gigalabichi1/procurement-management-system.git`
2. Install: `npm install`
3. Setup Database: `npm run db:setup`
4. Start: `npm run dev`

## ­ЪЊ▒ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў
- Frontend: React TypeScript
- Backend: Node.js Express
- Database: PostgreSQL
- Auth: JWT + bcrypt
